Welcome to MoRe!

MoRe is a platform where you can explore and discover information about movies, cast members, and user reviews. Whether you're a movie enthusiast or looking for recommendations, MoRe has something for everyone.

Search Movies, Cast, and Profiles

Use the search bar to find movies, cast members, and user profiles. Icons for each category are provided for easy identification.

Movies | Cast | Profiles

Movie Page Features

On a movie page, you can view the synopsis, reviews, average rating, and the number of user likes. Clicking on a username in a review allows you to see all reviews by that user. The cast list provides links to cast members' filmographies.

Create an Account

Creating an account enables you to log in, view your profile, leave reviews, and like movies to save them on your profile. User passwords are securely hashed and stored.
